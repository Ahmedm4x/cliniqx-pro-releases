## ✨ Features

### 📊 Dashboard
- Live statistics — today's appointments, total patients, revenue
- Weekly appointment chart updated in real time
- Today's schedule at a glance

### 📅 Appointments & Calendar
- Interactive monthly calendar
- Book appointments with time slot selection
- Confirm or delete appointments
- Auto-adds patients to records when booking

### 🦷 Dental Chart
- Interactive 32-tooth chart per patient
- Anatomically correct tooth shapes (incisors, canines, premolars, molars)
- Mark tooth conditions: Healthy, Cavity, Treated, Crown, Missing, Root Canal, Implant
- Add notes to individual teeth
- Saves automatically per patient

### 👥 Patient Management
- Complete patient records (name, phone, age, blood type, allergies)
- Treatment history with costs
- Edit and delete patients
- Search by name or phone number

### 💰 Price List
- Add/edit/delete services and products
- Prices in Iraqi Dinar (IQD)
- Category tags and filters
- Bilingual (English + Arabic)

### 🔐 Security & Licensing
- Secure account system per clinic
- 7-day free trial for new users
- License key activation
- Auto-expiry when trial or license ends
- Data isolated per clinic account

### ☁️ Cloud Sync
- Data synced to Firebase cloud
- Access from multiple devices with the same account
- Local backup — works offline too

## 🚀 Getting Started

1. **Download** the installer from the [Releases page](https://github.com/Ahmedm4x/cliniqx-pro-releases/releases/latest)
2. **Run** `CliniQX.Pro.Setup.1.0.3.exe`
3. **Install** — choose your installation folder
4. **Sign Up** with your email and clinic name
5. **Enjoy** your 7-day free trial — no credit card required
